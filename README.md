# Limpieza de Base de Datos de Asistencias

## Descripción

Este proyecto limpia una base de datos con registros duplicados e inconsistentes de asistencia a un curso

## Objetivo

Obtener una lista de participantes únicos y calcular su asistencia total

## Proceso

* Limpieza de correos electrónicos
* Estandarización de nombres
* Eliminación de valores inválidos
* Consolidación de registros duplicados
* Cálculo de asistencias

## Resultado

Se genera una base limpia con participantes únicos y se identifican aquellos con menor desempeño

## Archivos

* script.R
* Base para prueba.xlsx
* participantes\_limpios.csv

1. Abrir RStudio
2. Ejecutar el script
